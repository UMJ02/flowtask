# FlowTask v7.6.0-dashboard-mobile-refine-v7

## Cambios aplicados
- Se eliminó del workspace/dashboard el card de “Pizarra operativa”.
- Se redujo el tamaño visual de las métricas del hero y se compactaron sus cards.
- Se actualizó “Acciona sin rodeos” a “Inicia rápido” y se modernizó el bloque de accesos directos.
- En móvil, la vista de tareas ahora prioriza un bloque compacto de accesos rápidos con “Tareas +” y “Proyectos +”.
- En móvil se ocultaron “Busca tus tareas” y “Vistas rápidas de tareas”.
- La sección “Elige cómo quieres operar” pasó a “Elige tu vista”.
- Se eliminó la opción “Ambas”, dejando solo “Pizarra” y “Lista”.
- En móvil, el selector de vista quedó en dos columnas y más compacto.

## Base de datos
- No se inventaron columnas nuevas.
- La referencia de base real sigue siendo el esquema Supabase compartido por el usuario.
