# Flowtask V36 — Home Premium Video Intro

## Objetivo
Hacer que la vista de inicio se sienta más premium y más cercana al look del app.

## Cambios aplicados
- desktop:
  - layout dividido en dos:
    - izquierda: video introductorio
    - derecha: card de FlowTask con información y botones
- móvil:
  - no se muestra el video
  - el contenido se compacta para evitar scroll excesivo
- se refinó el card principal de información
- el bloque informativo de tres mensajes se volvió más interactivo:
  - rota cada 8 segundos
  - resalta un bloque a la vez
  - alterna presencia visual con una animación más limpia

## Resultado esperado
- inicio más premium
- mejor relación visual con el resto del app
- desktop más impactante
- móvil más limpio y usable
