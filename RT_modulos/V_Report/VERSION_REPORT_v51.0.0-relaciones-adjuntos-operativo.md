# Flowtask V51 — Relaciones + Adjuntos Operativo

## Objetivo
Cerrar mejor la operación entre módulos para que el usuario pueda trabajar con relaciones reales entre clientes, proyectos y tareas, además de usar adjuntos directamente en los detalles.

## Cambios aplicados
- tareas:
  - el detalle ahora incluye panel de adjuntos
- proyectos:
  - el detalle ahora incluye panel de adjuntos
  - se agregó acción rápida para crear una tarea vinculada al proyecto actual
- creación:
  - nueva tarea puede llegar con `projectId` y `clientName` precargados
  - nuevo proyecto puede llegar con `clientName` precargado
- clientes:
  - se agregaron CTA directas para crear:
    - proyecto para ese cliente
    - tarea para ese cliente

## Resultado esperado
- relaciones más fluidas entre módulos
- menos pasos manuales
- adjuntos disponibles en operación real
- mejor sensación de producto utilizable para cliente final
