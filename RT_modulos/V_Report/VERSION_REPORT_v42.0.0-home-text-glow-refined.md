# Flowtask V42 — Home Text Glow Refined

## Objetivo
Ajustar detalles finos del intro para mejorar legibilidad y dar más presencia visual al bloque rotativo.

## Cambios aplicados
- en desktop:
  - el título principal pasa a blanco
  - la descripción pasa a blanco suave
  - se agregan sombras sutiles de texto para mejor lectura sobre el video
- el card principal conserva transparencia pero con mejor lectura sobre fondo oscuro
- los textos rotativos ahora tienen un tratamiento más actual:
  - glow sutil por color
  - reborde luminoso
  - microanimación de pulso
  - transición suave entre estados

## Resultado esperado
- mejor contraste en desktop
- look más premium
- mensajes rotativos más llamativos sin verse sobrecargados
