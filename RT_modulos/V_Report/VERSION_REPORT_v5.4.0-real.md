# FlowTask v5.4.0-real

Base real ejecutable para continuar sin perder scripts ni estructura de Next.js.

## Qué corrige
- Recupera `package.json` completo con scripts reales: `dev`, `build`, `start`, `lint`, `typecheck`.
- Conserva la línea de `Workspace` e `Intelligence` ya integrada.
- Mantiene la redirección de `/app/dashboard` hacia `/app/workspace`.
- Mantiene sidebar, mobile nav y command palette alineados con la re-arquitectura.

## Scripts disponibles
- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run typecheck`

## Base usada
- Proyecto real sobre la línea técnica estable, no scaffold aislado.
