# Flowtask V18 — Board Panels Under Header

## Objetivo
Mover el despliegue de paneles para que aparezca debajo del bloque superior de “Modo pizarra”, manteniendo el botón de mostrar/ocultar y sin usar panel flotante lateral.

## Cambios aplicados
- El panel de módulos vuelve a ser inline.
- Ahora se despliega justo debajo del header de pizarra.
- Se le dio un estilo distinto en tonos azul/suaves para diferenciarlo del contenido principal.
- Se mantiene el botón de mostrar/ocultar paneles.
- Se agregó botón de cerrar dentro del bloque desplegado.

## Resultado esperado
- La información principal no se compacta lateralmente.
- El usuario entiende mejor que es un bloque de configuración temporal del tablero.
- La interacción queda más limpia y más predecible.
