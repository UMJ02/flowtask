# Go Live Gate v9

## Solo considerar salida cuando:
- No existan bugs críticos abiertos
- Los bugs altos estén resueltos o explícitamente aceptados
- Login, dashboard, tareas y proyectos estén validados
- Build complete sin ruptura
- Variables de entorno productivas estén configuradas
- Endpoints de salud respondan correctamente
- Se haya hecho al menos una prueba móvil básica

## No marcar como listo si:
- CRUD de tareas falla
- CRUD de proyectos falla
- Auth redirige mal
- Existen errores runtime visibles
- La navegación principal pierde contexto
