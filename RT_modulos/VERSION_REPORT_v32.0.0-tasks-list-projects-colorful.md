# Flowtask V32 — Tasks List + Colorful Projects

## Objetivo
Simplificar la vista de tareas eliminando la lógica de pizarra/flujo de esa pantalla y llevarla a una lista accionable. Además, hacer que los cards de proyectos se diferencien mejor visualmente.

## Cambios aplicados

### Vista tareas
- se eliminó la vista de:
  - zona de trabajo
  - flujo
  - pizarra
- se reemplazó por una lista de tareas con:
  - nombre
  - status
  - deadline
  - ver
  - editar
  - borrar
  - botón de finalizar
- al finalizar:
  - la tarea anima salida
  - pasa a la lista de concluidas
- se agregó botón:
  - `Ver tareas concluidas`

### Vista proyectos
- se mantuvo la estructura
- se volvieron más llamativos los cards con acentos de color diferenciados

## Resultado esperado
- tareas mucho más directas y entendibles
- proyectos más fáciles de distinguir visualmente
