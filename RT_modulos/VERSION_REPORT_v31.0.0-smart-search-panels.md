# Flowtask V31 — Smart Search Panels

## Objetivo
Hacer que la búsqueda de tareas y proyectos sea más limpia y menos pesada visualmente, dejando los filtros avanzados ocultos hasta que de verdad se necesiten.

## Cambios aplicados
- se reemplazó el bloque visible completo de filtros por una barra de búsqueda principal
- se agregó botón de:
  - `Búsqueda avanzada`
- los filtros avanzados se despliegan solo al abrirlos
- se dio un estilo más fino y más claro a los formularios de búsqueda
- se aplicó en:
  - tareas
  - proyectos

## Resultado esperado
- interfaz más simple
- menos ruido visual
- búsqueda más entendible
- filtros disponibles sin ocupar toda la pantalla desde el inicio
