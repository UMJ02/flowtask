# Flowtask V8 Release Candidate

## Nuevos comandos
- `npm run release:check`
- `npm run release:bundle`

## Ruta sugerida
1. `npm install`
2. `npm run qa:full-client`
3. `npm run ux:review`
4. `npm run release:check`
5. `npm run build`

## Propósito
Esta versión busca cerrar una salida más controlada y más cercana a entrega cliente.
