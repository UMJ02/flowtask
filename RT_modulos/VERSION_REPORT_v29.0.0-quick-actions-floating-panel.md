# Flowtask V29 — Quick Actions Floating Panel

## Objetivo
Separar el panel de acciones rápidas del card principal del hero para que no se corte ni quede visualmente atrapado dentro de ese contenedor.

## Cambios aplicados
- el botón `Acción rápida` sigue inline junto a los otros botones
- el panel ahora es una capa flotante independiente
- aparece al hacer click y entra desde la derecha
- ya no queda contenido dentro del mismo espacio del card grande
- se mantiene la X de cierre y el overlay suave

## Resultado esperado
- panel visualmente limpio
- sin cortes
- comportamiento flotante real e independiente
