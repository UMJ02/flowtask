# Flowtask V6 QA

Esta versión agrega una capa de QA funcional para acercar el proyecto a cliente real.

## Comandos nuevos
- `npm run qa:functional`
- `npm run qa:report`
- `npm run qa:full-client`

## Secuencia recomendada
1. `npm install`
2. `npm run qa:smoke-local`
3. `npm run qa:functional`
4. `npm run qa:report`
5. `npm run build`
