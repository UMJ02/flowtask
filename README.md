# FlowTask

Versión actual: `5.1.0-workspace-real`

## Dirección del producto
FlowTask ahora está orientado a una experiencia más clara:
- **Workspace** como entrada principal
- **Intelligence** como lectura unificada del estado del trabajo
- menos ruido en navegación
- textos más humanos y cortos

## Navegación principal
- Workspace
- Projects
- Tasks
- Clients
- Intelligence
- Reports
- Settings

## Scripts
- `npm run dev`
- `npm run build`
- `npm run typecheck`

## Nota de arquitectura
Las pantallas anteriores como onboarding, planning, control tower, risk radar, execution center, workspace OS y executive suite quedaron fuera de la navegación principal para evitar saturación. La intención es consolidar su valor dentro de Intelligence y mantener Workspace como centro operativo real.
