# Flowtask V49 — Mobile Menu Compact + Profile Links

## Objetivo
Afinar la experiencia móvil para que el drawer no tenga tanto aire azul sobrante y para que perfil/notificaciones funcionen como accesos directos.

## Cambios aplicados
- menú móvil:
  - ya no ocupa visualmente toda la altura con aire sobrante
  - ahora queda más compacto con altura máxima controlada
  - se agregó `Cerrar sesión` como item del menú móvil
- perfil en móvil:
  - ya no despliega dropdown
  - ahora redirige directo a `/app/settings`
- desktop:
  - se conserva el dropdown normal del perfil
- notificaciones:
  - se mantiene como acceso directo por link

## Resultado esperado
- drawer móvil más limpio
- menos espacio vacío
- accesos móviles más directos y funcionales
