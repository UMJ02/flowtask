# FlowTask v2.3.0-execution-center

## Objetivo
Crear una capa de ejecución que convierta señales del workspace en acciones concretas, sin romper la base v2.2.0.

## Incluye
- nueva ruta `/app/execution-center`
- nuevo query agregador `getExecutionCenterSummary()`
- execution score
- frentes `Do now`, `Unblock` y `Monitor`
- team pulse por departamento
- widget compacto en dashboard
- acceso desde sidebar y command palette
- salida imprimible `type=execution`
- `README.md` y `package.json` actualizados

## Validación
- `npm run typecheck` en cero
