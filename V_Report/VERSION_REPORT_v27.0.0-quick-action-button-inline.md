# Flowtask V27 — Quick Action Button Inline

## Objetivo
Quitar la cejilla deslizable y mover el disparador de acciones rápidas a la misma línea del botón “Abrir pizarra”.

## Cambios aplicados
- se eliminó el `button.group.absolute` de la cejilla deslizable
- se creó un botón inline:
  - `Acción rápida`
- el botón queda en la misma línea que:
  - `Abrir pizarra`
- el card de accesos rápidos se despliega debajo de esa fila
- el card ahora tiene una `X` de cerrar alineada a la derecha del encabezado, en línea con el pill `Accesos rápidos`

## Resultado esperado
- interacción más clara
- menos ruido lateral
- acceso rápido más integrado al hero principal
