# Flowtask V33 — Header Refined Green

## Objetivo
Quitar el efecto de doble card del header y darle una presencia más limpia y más pro, usando mejor la paleta verde del sistema.

## Cambios aplicados
- se eliminó el wrapper extra del header para evitar el efecto de doble card
- el header quedó como una sola pieza más limpia
- se aplicó una presencia más premium con:
  - borde suave verde
  - toque de gradiente radial en verde
- el círculo del ícono de la app ahora es completamente negro para que la marca se vea más sólida y redonda

## Resultado esperado
- header más limpio
- menos ruido visual
- mejor integración con la paleta del sistema
- ícono de app más elegante
