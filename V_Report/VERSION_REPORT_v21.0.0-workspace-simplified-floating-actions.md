# Flowtask V21 — Workspace Simplified + Floating Actions

## Objetivo
Hacer el workspace mucho más simple, reduciendo métricas repetidas y moviendo los accesos rápidos a un elemento flotante menos invasivo.

## Validación UX realizada
Se detectó repetición entre:
- Workspace overview
- Board overview
- Client metrics
- Collaboration metrics
- User workload
- Department metrics

Estas capas repetían lectura ejecutiva sin aportar suficiente valor para un usuario que quiere actuar rápido.

## Cambios aplicados
- Se mantiene **Workspace overview**.
- Los **Accesos rápidos** salen del flujo principal.
- Se crea un elemento flotante lateral derecho con:
  - cejilla animada
  - mensajes como:
    - “A un click de empezar”
    - “Hazlo rápido”
    - “Entra y sigue”
- El dashboard principal se simplifica a:
  - Workspace overview
  - Salud de proyectos
  - Proyectos a vigilar
  - Actividad reciente

## Resultado esperado
- menos ruido visual
- menos métricas duplicadas
- lectura más simple
- acciones rápidas disponibles sin ensuciar el dashboard
