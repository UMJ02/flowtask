# Flowtask V22 — Floating Tab Center + Color Pulse

## Objetivo
Refinar la cejilla del card flotante de accesos rápidos para que se vea más centrada, más limpia y menos cambiante.

## Cambios aplicados
- la cejilla ahora queda centrada verticalmente respecto al card
- se eliminó la rotación de varios textos
- se deja un solo mensaje:
  - `A un click de empezar`
- en lugar de cambiar texto, la cejilla alterna presencia visual con dos tonos:
  - azul cuando está cerrada
  - verde cuando está abierta
- se mantiene la animación flotante para darle vida sin meter ruido

## Resultado esperado
- elemento más claro
- mejor lectura
- más premium y menos cargado
