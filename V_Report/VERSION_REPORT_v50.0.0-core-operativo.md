# Flowtask V50 — Core Operativo

## Objetivo
Dar el primer paso real de cierre hacia producto cliente final, priorizando los módulos core.

## Cambios aplicados
- navegación reducida al core operativo
- creación de páginas reales para:
  - detalle de tareas
  - edición de tareas
  - creación de tareas
  - detalle de proyectos
  - edición de proyectos
  - creación de proyectos
  - detalle de clientes
- módulo de clientes reforzado con:
  - crear
  - editar
  - borrar
  - ver detalle
- sidebar desk alineado con navegación core

## Resultado esperado
- menos rutas sin valor visible
- core más usable
- mejor continuidad de navegación
- primera base seria para seguir con V51
