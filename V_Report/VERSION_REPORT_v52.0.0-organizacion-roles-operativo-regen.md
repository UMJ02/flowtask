# Flowtask V52 — Organización + Roles Operativo (regen)

## Objetivo
Hacer usable la capa de organización para que el cliente pueda gestionar miembros, roles e invitaciones dentro del producto.

## Cambios aplicados
- organización:
  - la página ya carga miembros reales de la organización activa
  - mantiene métricas, roles, permisos e invitaciones en una vista más operativa
- miembros:
  - ahora se pueden editar roles directamente desde la lista
- invitaciones:
  - el formulario crea invitaciones directo con Supabase autenticado
  - la bandeja permite revocar invitaciones pendientes
- navegación:
  - se agregó `Equipo` al menú principal
  - quedó alineado en desk y móvil
