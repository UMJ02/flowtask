# FlowTask v6.3.0-board-layout-fix

## Base
Se tomó como base el ZIP subido por el usuario y se añadió la vista opcional de pizarra blanca en `/app/dashboard?view=board`.

## Cambios
- Nueva vista opcional de pizarra blanca.
- Notas y recordatorios movidos fuera del contenedor de paneles para evitar sobreposición.
- Calendario compacto con vista de 2 semanas y vista mensual.
- Paneles removibles y reactivables desde el aside.
- Botón Pizarra en el workspace para abrir la vista visual.

## Nota
El proyecto base aún contiene errores de tipos en otros módulos no relacionados con esta pasada.
