# Flowtask V48 — Mobile Menu Match Desktop

## Objetivo
Hacer que el menú móvil muestre únicamente los mismos botones principales que ya existen en la versión desktop.

## Cambios aplicados
- el menú móvil ahora usa la misma lógica de navegación principal del sidebar desktop
- se removió la sección extra de botones que no aparecen en el menú lateral desktop
- se conserva el diseño del drawer móvil y su comportamiento con portal

## Resultado esperado
- menú móvil más limpio
- misma navegación principal que desktop
- menos botones sin uso visible por ahora
