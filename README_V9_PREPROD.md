# Flowtask V9 Preproduction

## Nuevos comandos
- `npm run preprod:validate`
- `npm run bugs:template`
- `npm run preprod:bundle`

## Ruta sugerida
1. `npm install`
2. `npm run qa:full-client`
3. `npm run ux:review`
4. `npm run release:check`
5. `npm run preprod:validate`
6. `npm run bugs:template`
7. `npm run build`

## Propósito
Cerrar una revisión de preproducción con checklist y registro de bugs antes de una salida final.
