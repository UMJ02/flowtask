# V58.10.9 – Organization management UI refine

## Objetivo
Pulir la vista de organización para que se sienta coherente con el resto de FlowTask y agregar acciones administrativas básicas sin rehacer estructura.

## Cambios clave
- ajuste visual del card superior de organización
- textos más humanos dentro del módulo organización
- paleta alineada a los tonos base de la app
- permisos por cliente e invitaciones con lectura desplegable
- bitácora desplegable con flecha
- nuevo card administrativo para editar nombre, salir o eliminar organización

## Base
Esta versión toma como base directa la **V58.10.8**.
