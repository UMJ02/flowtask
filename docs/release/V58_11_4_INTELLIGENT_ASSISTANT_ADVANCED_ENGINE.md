# V58.11.4 – Intelligent Assistant Advanced Engine

## Base
Esta versión se construye sobre V58.11.3.

## Objetivo
Subir el asistente inteligente a una segunda etapa con mayor criterio, más reglas, memoria por workspace y una interfaz más limpia.

## Cambios clave
- Motor avanzado del asistente con scoring de prioridad.
- Nuevas señales: stale tasks y tareas sin fecha.
- Configuración avanzada con reglas por tipo de alerta.
- Memoria por workspace para silencios y cierres.
- Máximo configurable de cards visibles.
- Pulido visual general con cards más compactos y menos texto.
